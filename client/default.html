Devbench

Write some code in the left window. Use devbench::submit_result to output the result. Supported overloads:

  • devbench::submit_result(std::string const&)
  • devbench::submit_result(std::span<std::pair<float, float>> const&, devbench::table_tag)
  • devbench::submit_result(std::span<std::pair<float, float>> const&, devbench::curve_tag)
